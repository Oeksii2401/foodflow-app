# FoodFlow

Мультиязычный сервис заказа еды для кафе с автоматическим разделением платежей через Stripe Connect.

## Стек
- **Backend**: FastAPI + Python
- **Frontend**: HTMX + Jinja2 + Tailwind
- **БД**: PostgreSQL
- **Деплой**: Railway

## Быстрый старт

```bash
pip install -r requirements.txt
uvicorn main:app --reload
